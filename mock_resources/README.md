# Система аутентификации и авторизации

## Компоненты системы
1. Пользователи: регистрация, вход, выход, обновление профиля
2. Система разграничения прав доступа
3. Mock-ресурсы для демонстрации

## Схема прав доступа
- Каждый пользователь может иметь набор разрешений
- Разрешения определяются связью:
  - Пользователь
  - Ресурс
  - Действие (READ, WRITE, UPDATE, DELETE)

## Технологический стек
- Django Rest Framework
- PostgreSQL
- JWT-аутентификация
